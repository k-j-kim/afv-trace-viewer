# Agentforce Trace Viewer

A single-page viewer for `trace.md` files generated by Agentforce Vibes.

**Live app:** https://k-j-kim.github.io/afv-trace-viewer/

## Usage

1. Open the app in your browser.
2. Paste the contents of your `trace.md`, drag & drop the file, or upload it.
3. Click **Parse Trace** to explore the conversation turns.

## Features

- Stats bar: Turns, Tokens In/Out, Cache Read, Total Tokens, Avg TTFT, Elapsed
- Per-turn detail: model, token counts, TTFT, tools called, skills used, task progress
- Formatted and raw views for full request/response bodies
- Collapsible turns with search/filter
- Share traces via URL using a public GitHub Gist raw URL (`?url=<raw-url>`)

## Trace format

Traces are Markdown files with one section per turn, separated by `---`. Each turn is produced by the Agentforce Vibes extension for Claude Code.
